# Archidemus GitHub Pages

Sitio web personal en GitHub Pages. Astro + TailwindCSS + shadcn/ui.

## Stack

- Runtime: `bun`
- Framework: Astro + TailwindCSS
- Componentes: shadcn/ui

## Comandos

```bash
bun install
bun run dev
bun run build
```

## Deploy

Deploy automatico via GitHub Actions al hacer push a `main`.
